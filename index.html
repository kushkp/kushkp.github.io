Ruby-based command line version of the classic board game

Players may use the WASD keys to navigate a cursor across the board. Validation algorithms display available moves, including highlighting possible capture moves. Player cannot make naive moves that result in player's own "check". Multi-level class inheritance structure DRYs up code.
